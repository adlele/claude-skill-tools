# Concepts

Core concepts that power composer.

| Concept                      | Description                                                                                  |
| :--------------------------- | :------------------------------------------------------------------------------------------- |
| [Compositions](compositions) | Multi-step pipelines that orchestrate AI agent workflows from planning through PR creation.  |
| [Roles](roles)               | Markdown system prompts that define each agent's behavior, boundaries, and output format.    |
| [The Ralph Loop](ralph)      | The automated developer/reviewer iteration cycle that drives code toward a clean review.     |
| [Sandboxes](sandboxes)       | Isolated git worktrees that keep agent work separated from your main repository.             |
| [Distill](distill)           | Synthesis of planning artifacts into a zero-ambiguity feature request for automated re-runs. |
| [Retro](retro)               | Extraction of reusable learnings from ralph review iterations into persistent guidance.      |
