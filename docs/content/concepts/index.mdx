# Concepts

Core concepts that power composer.

| Concept                      | Description                                                                                  |
| :--------------------------- | :------------------------------------------------------------------------------------------- |
| [Compositions](/concepts/compositions) | Multi-step pipelines that orchestrate AI agent workflows from planning through PR creation.  |
| [Roles](/concepts/roles)               | Markdown system prompts that define each agent's behavior, boundaries, and output format.    |
| [The Ralph Loop](/concepts/ralph)      | The automated developer/reviewer iteration cycle that drives code toward a clean review.     |
| [Sandboxes](/concepts/sandboxes)       | Isolated git worktrees that keep agent work separated from your main repository.             |
| [Distill](/concepts/distill)           | Synthesis of planning artifacts into a zero-ambiguity feature request for automated re-runs. |
| [Retro](/concepts/retro)               | Extraction of reusable learnings from ralph review iterations into persistent guidance.      |
