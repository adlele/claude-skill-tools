# Cookbook

Practical recipes for common workflows. Each recipe is a step-by-step guide.

| Recipe | Description |
|:-------|:------------|
| [Your First Feature](/cookbook/first-feature) | Walk through a full composition from feature request to pull request |
| [Iterate with Ralph](/cookbook/iterate-with-ralph) | Use the ralph-only composition for well-defined tasks with developer/reviewer loops |
| [Distill & Re-run](/cookbook/distill-rerun) | Refine ambiguous results by distilling a better feature request and re-running |
| [Custom Roles](/cookbook/custom-roles) | Override shipped prompts, create new roles, and use the --idea flag |
| [Headless Compositions](/cookbook/headless-batch) | Run compositions in the background for fire-and-forget execution |
| [Metrics & Cost Tracking](/cookbook/metrics-and-cost) | Generate reports, explore sessions, and understand cost drivers |
